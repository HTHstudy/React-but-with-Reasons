# Why-Driven Development

왜(Why)에 집착하는 개발자입니다.  
프론트엔드 아키텍처를 설계할 때, 코드보다 먼저 "왜 이렇게 해야 하는가"를 묻습니다.  
React를 사용하며 내리는 결정들과 그 이유를 기록합니다.
